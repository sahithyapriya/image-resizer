# image-resizer (800x600)

This is a Python-based image resizer tool that processes all images in the `input_images/` folder and resizes them to exactly 800Ã—600 pixels. The resized images are saved in `output_images/`.

## How to Use

1. Place your images inside the `input_images/` folder.
2. Run the script:
   python resizer.py
